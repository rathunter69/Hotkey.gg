Unhide the rows

Hidden rows lie — unhide them, then group so the detail stays honest.
The last analyst HID the monthly detail instead of grouping it — rows 4 to 7 are just… gone. No ⊞, no rail, no trace: the row numbers skip and the subtotal reads like magic. The MD wants it done RIGHT. Select across the gap and unhide (Ctrl+Shift+9 — or Alt H O U O), group the detail properly (Shift+Alt+→), fold it back (Alt A H) so the ⊞ tells the next reader where it lives — and while you’re in there, the Size column is squeezed to hash marks: set it to a real width (Alt H O W).

The shortcuts in this drill
Alt walks: tap Alt, release, then the letters in sequence — each key steps one ribbon menu. On a Mac, tap ⌥ and use the same letters (KeyTips, Excel 2024+).
Ctrl+Shift+9⌘⇧9 on macUnhide rows
Unhide rows an export or a previous owner collapsed — hidden detail is where errors hide too.
Alt+Shift+→⌥⇧→ on macGroup rows/columns (outline)
Group detail rows so a reviewer can collapse to the summary — finished models ship with the detail folded.
Alt a h⌥ a h on macHide (fold) group detail
Collapse grouped detail to read the summary — the fold hides rows without deleting them.
Alt h o w 1 2⌥ h o w 1 2 on macSet column width…
Exact column widths when autofit gives a ragged page — uniform width is part of house style.
