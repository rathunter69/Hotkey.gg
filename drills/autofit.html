Fix the squeezed columns

Alt H O I autofits to content; a print block wants ONE width — set it with Alt H O W.
Two problems, two different fixes — and knowing which is which is the skill. The label block is squeezed to hash marks: Alt H O I fits it to its contents. The four data columns are RAGGED — every width different, and autofit would keep them uneven. Select across them and set ONE width: Alt H O W, 12, enter. Autofit is for content; uniformity is for print.
