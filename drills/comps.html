Run the comps

Build one row of logic, then Ctrl+D stamps it down the page — and LARGE/SMALL read the range with the outliers trimmed off.
A real comps page, both panels: compute EV/EBITDA for five peers and fill it down, read the data — median, high, low, then the TRIMMED read (2nd-high and 2nd-low, because one weird peer shouldn’t set your range) — then walk the target through the implied chain: EV, equity, per share, premium to market. The multiples aren’t given anymore; you build them.
