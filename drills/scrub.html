Clean the export

Exports come dirty — delete junk rows (Ctrl+−), sort (Alt A S), re-total; the data is yours.
The deal list came out of the system DIRTY: a duplicated header row, a "--- PAGE 2 ---" break, and a stale SUBTOTAL row that will double-count the moment anyone totals the column. Scrub it — delete all three junk rows, sort what is real largest-first, and re-total the clean data. Every export you will ever receive needs this pass.

The shortcuts in this drill
Alt walks: tap Alt, release, then the letters in sequence — each key steps one ribbon menu. On a Mac, tap ⌥ and use the same letters (KeyTips, Excel 2024+).
Alt h d r⌥ h d r on macDelete rows
Delete full rows as structure changes — live formulas rewire themselves around the edit.
Alt a s d⌥ a s d on macSort descending (Data tab)
Sort the tape before reading it — biggest positions first is how a book gets reviewed.
Alt+=⌥ = on macAutoSum the adjacent range
AutoSum reads the block above or beside the cursor and proposes the SUM — foot rows and columns without typing a formula.
Ctrl+B⌘B on macBold
Bold marks structure: titles, total rows, anything the reviewer’s eye should land on first.
