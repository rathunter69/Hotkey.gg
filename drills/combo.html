Clean the paste

Select the whole range, format once — Alt H chords work a block, not a cell.
A comp sheet came over as a raw paste: title with no weight, headers slumming it lowercase-left, figures unformatted, the notes column overrunning its neighbors, half the columns showing ####. Make it presentable end to end — which metrics came over, and where the block sits, changes every run.
train this drill → par 22 keystrokes · keyboard only · free
The keys
A1 + headers A2:E2 → ctrl+b · metric headers B2:D2 → alt h a r · figures B3:D6 → alt h k + alt h 9 ×2 · notes E3:E6 → alt h w · then alt h o i — no #### ships
Step by step
  1. weight first: A1 · ctrl+b · then the header row A2:E2 · ctrl+b
  2. numbers read right-aligned — headers over them too: B2:D2 · alt h a r
  3. figures to commas: B3:D6 · alt h k · alt h 9 twice
  4. the notes column wraps instead of spilling: E3:E6 · alt h w
  5. last pass, always: select the number columns · alt h o i — nothing ships showing ####
