Columns move too

Ctrl+Space selects the whole column — Alt H I C inserts one, Alt H D C deletes one.
Fix the quarterly schedule’s columns: delete the stale draft column and insert the missing quarter. Ctrl+Space selects the whole column, Alt H D C deletes it, Alt H I C opens the gap where the missing quarter belongs — then add the new column’s header and format it to match.

The shortcuts in this drill
Alt walks: tap Alt, release, then the letters in sequence — each key steps one ribbon menu. On a Mac, tap ⌥ and use the same letters (KeyTips, Excel 2024+).
Ctrl+Space⌘Space on macSelect the entire column
Select the whole column — the setup move for inserting, deleting, or formatting column-wide.
Alt h d c⌥ h d c on macDelete columns
Delete full columns cleanly — references adjust, nothing dangles.
Alt h i c q 2⌥ h i c q 2 on macInsert columns
Insert columns for the new quarter — the model makes room without re-keying.
Ctrl+B⌘B on macBold
Bold marks structure: titles, total rows, anything the reviewer’s eye should land on first.
