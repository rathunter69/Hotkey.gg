Dress the tab — full formatting pass

A full dress pass is chords end to end — bold, comma, percent, border, shade, blue, italic — and EBITDA margin is EBITDA over the Revenue line right above it.
A finished analysis, zero formatting — the state every tab is in at 3:58am. Dress it like it’s going in the book: title bolded and ruled, hardcodes in blue, EBITDA margins as clean percents, dollars with commas, the output circled, and the source line whispering in italics. Seven formatting muscles, one pass, and the block moves every run.

The keys
A1 bold + alt h b b · header A3:E3 → alt h h gray · blue the hardcodes B4:E7 + B9:E9 → alt h f c · EBITDA margins row 11 → ctrl+shift+% +1 dec · figures B4:E10 → alt h k · circle the OUTPUT A11:E11 → alt h b o · footnote A11 → ctrl+1 e · EBITDA row → ctrl+b + alt h b t · source A13 → ctrl+i
Step by step
  1. the title · A1 · ctrl+b then alt h b b — bold, and a rule underneath
  2. shade the header band — select A3:E3 · alt h h · → Gray (header) · ↵ — a shaded header reads as a header at a glance
  3. blue the hardcodes — the typed numbers, not the computed ones: select B4:E7 (segment revenues) · alt h f c · →×4 Blue · ↵ · then the cost line B9:E9 the same way — blue = typed, black = computed
  4. the EBITDA margins — B11:E11 (EBITDA ÷ the Revenue line above) · ctrl+shift+% · alt h 0 — one decimal
  5. comma the dollar figures — select B4:E10 · alt h k — thousands separators; negatives flip to (parens)
  6. circle the OUTPUT — select the margin row A11:E11 · alt h b o — one perimeter around the row the MD reads
  7. footnote the margin label — on A11 · ctrl+1 e — superscript stamps the ¹
  8. the EBITDA line carries the page — select A10:E10 · ctrl+b · alt h b t — bold with a rule above
  9. the source line whispers — on A13 · ctrl+i — italic marks a footnote/source
