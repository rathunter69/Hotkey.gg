## PART I — SYSTEM MAP

### 1. Product scope & flow
Keyboard-only Excel trainer (browser Excel emulator; timed drills, real hotkeys, no mouse). Audience: finance/IB/PE analysts, finance clubs, recruiting candidates. Two ladders: **LEVEL** (volume, never resets) + **RANK** (competitive, opt-in at Level 10). Model: freemium B2C (PRO $7/mo · $19/season, `drills.js:497`) + B2B "PRO for the desk" + free comped PRO for student clubs. **Currently private beta:** PRO free for all (`BETA_MODE=true` `index.html:1796`), payments not live (Stripe test scaffold), new devices gated by an access-code curtain (`PRELAUNCH_LOCK` `index.html:1801`).

Pages: `index.html` (THE trainer — engine + everything, 15.9k lines/1.1MB), `leaderboard.html`/`desks.html` (boards via `lb.js`), `stats.html`, `profile.html` (+card customizer), `account.html`, `billing.html` (scaffold), `reference.html` (cheat-sheet), `About.html`/`enterprise.html`/`contact.html` (marketing/sales), `cert.html` (verifiable certs), `admin.html` (ops), legal pages, `drills/*.html` (82 SEO pages).

Journeys (code): landing `index.html:1766` → curtain `showPrelaunchLock:15437` → enter `tryEnter:11266` → onboarding `showKeyboard→showComfort→runTour:15238` → play (guest runs NOT saved `:9524`) → signup `nav.js:999` → level `hkLevelXp:13746` → ranked nudge `nav.js:336` → PRO `hkProSheet nav.js:1397` → certs `maybeOfferCertificate:9326` → desks `account.html:305`.

### 2. Drills & curriculum
Two-file split (`drills.js:1-16`): **drills.js** = canonical catalog metadata + progression/reward config (`window.HOTKEY_DRILLS:23`; groups[], meta{}, derived menuOrder/groupOf/labelOf); **index.html** = per-drill engine logic (`CHALLENGES` `:1889`). Trainer overrides name/label from drills.js at runtime. Verified: 8 groups, **82 drills**, menuOrder===82.

Drill model: `{name,label,par,parKeys,aha,prompt,req(),guide(),targets(),demo(),build(),checks(S)}`. Hint ladder req→guide→aha→demo. Grading is **end-state** (`checks(S)` reads live cells; Freedom Doctrine = grade WHAT not HOW, `DRILL_DOCTRINE.md:54`). Chapters: Foundations(10)→Formatting(10)→Formulas I(11) [FREE] → Data&Lookups(10, bridge) → Formulas II(11)/Models I(10)/Models II(10)/Full Builds(10) [PRO]. Cert tracks `HK_TRACKS:175` (fluency/formulas/modeling). Landing pages generated by `dev/build-drill-pages.js` (launches real trainer, mines `demo()` for chords) + CI drift guard (`gate.yml:90`).

### 3. Spreadsheet engine (index.html)
`blankCell():1880` flat schema; global `S:14345`; sparse `"A1"` hash; grid A..J (10 cols), `__ROW_CAP=20`. `recalc():9882` = fixed-point **max 8 passes**, no cycle detection. `evalFormula():7912` hand recursive-descent parser; functions include SUM/IF/INDEX/MATCH/OFFSET/CHOOSE/SUMIF(S)/IRR/NPV/text fns — **no VLOOKUP/HLOOKUP, no AND/OR/NOT, no date math**. Clipboard `copySel:10777`/`doPaste:10706` (all/values/formulas/formats/transpose + paste-op arithmetic). Ribbon `TABS:8691`/`MENUS:8700`/`applyRibbon:10496` (border keys Excel-canon since r298). Undo depth 60; `commitAction:9975` universal hook.

Modes (all ride `C.demo`+`C.checks`): guided/rails (`S._railZone` fence + `railSafe` skips cursor-graded drills), hints/F1 (free cursor), rapid-fire (real grid, matcher scorer), demo-replay, echo (never scores). Grid-sizing (`render:8342`) is the **most-churned, fragile** area — elastic width + fixed 26px height + `__ROW_CAP` + ResizeObserver, three measurement fallbacks. `CHARPX=8.6` magic constant couples display-width + `####` overflow + grading.

### 4. Cosmetics / cards / progression (themes.js, nav.css)
One unified card `hkPlayerCard(themes.js:1996)` everywhere. Layers: skin (`hkFlair:1081`→`hk-frame-<id>`→`hkFrameOrnaments`), title notch, rank crest/tier (passed pre-rendered), LVL tray. Engines `HK_RANK:662` (8 TIERS, `levelOf:716`, `computeXP:726` v4 trading-day decay) — SSOT after consolidating 6 level + 4 XP copies. Unlock gating `hkFrameEarned:898` (tier/level/streak/cert/perfect/chapter/charter), beta unlock-all `hkBetaUnlocked:890`. fx `hkInitCardFx:1366` = one shared rAF loop, 1 canvas/card, reduced-motion static, **class-scaled EL density (r414)**. Notch taxonomy r414 `nav.css:708` (angular status+achievement+onyx+diam / rounded ladder base).

### 5. Security & backend (Supabase + Stripe + Cloudflare)
37 idempotent migrations. Auth = Supabase Auth (email/pw + anon). **AuthZ entirely in Postgres RLS + SECURITY DEFINER RPCs on `auth.uid()`** — client holds only the public publishable key. RLS unusually disciplined: column-frozen `profiles` grants, `runs` refereed by `runs_guard` trigger (rejects impossible times, shadow-flags implausible, flag not client-settable), `entitlements` service-role-only, invite codes never client-readable, PRO computed by `my_pro()` (not a client flag), all definers pin `search_path`. **XSS reviewed clean** (esc()/escHtml() everywhere; handle server-regex-constrained). **No secrets committed** (only the public anon key). Edge fns: `create-checkout` (Stripe test), `weekly-digest`.

### 6. Pipeline / CI / deploy
`gate.yml`: PR + push-to-main; **scope-gated** — engine matrix (parity 124, alt-paths 74, guided/replay ~82, onboard 36, mac-input 19, rapidfire 15 + drill-page drift guard) runs only when `index.html|drills.js|dev/|.github/` changed; else cosmetic lane (smoke 7 + lb 36) ~1min. `supabase-deploy.yml` (separate; push to main on `supabase/**`; manual live smoke `smoke-live.mjs`). Cloudflare Pages deploys `main`. Cache-bump `?v=` across all HTML is the **only** cache-bust (no service worker), **manual and unenforced**. Cadence: build → local gate (:8791) → cache-bump → PR → merge on green → restart branch off main (force-with-lease).

## PART II — CROSS-CUTTING THEMES (the patterns that recur)

1. **Single-source-of-truth drift.** The same fact is hand-copied across many places and drifts: drill-membership in 4 spots + `HOTKEY_PARS` (a 5th) [#76]; skin definitions across `HK_FRAMES`/`SKINS`/notch-lists/border-spin-list [caused the #121 pill bug]; clipboard format-field list in ≥4 spots [caused the r407 fill bug]. **Pattern fix: derive, don't duplicate; where you must duplicate, assert equality in the gate.**
2. **Critical discipline that is manual and unenforced.** Cache-bump `?v=` (silently blocked live delivery for 3 rounds, r390-392); de-hinting drill copy; drilling the DB deploy by hand. **Pattern fix: move each to a CI assertion.**
3. **No verification of the things that actually break.** No visual-regression for the cosmetic lane (which carries most change traffic); no check that the deployed Cloudflare/Supabase artifact matches the repo; backend RLS proven only by a manual smoke. **Pattern fix: cheap external post-deploy + asset-consistency checks.**
4. **Accumulated dead code from fast iteration.** Whole unreachable ornament machinery + ~8 dead fx generators (art reworks left the old code in place). **Pattern fix: scheduled subtraction passes; smoke covers it.**
5. **Single 15.9k-line global scope (index.html).** "A cosmetic edit can break every drill" — yet shared-script edits (themes/nav) skip the engine matrix. **Pattern fix: widen the cosmetic lane OR reclassify shared scripts as engine-scope.**
